# DOODBA ODOO 14

## Despliegue entorno desarrollo

**Descargar y construir imagen**


```bash
docker-compose -f devel.yaml build --pull --no-cache
```

**Descargar y construir imagen**
```
invoke git-aggregate
```

**Iniciar**
```
invoke start
```

**Parar**
```
invoke stop
```
